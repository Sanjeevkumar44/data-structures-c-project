# Call Center Queue Management System (Data Structures in C)

## Project Description
This project demonstrates the implementation of a Queue data structure using the C programming language. 
It simulates a simple Call Center Queue Management System where incoming calls are added to a queue and 
handled in First-In-First-Out (FIFO) order.

## Objective
The main objective of this project is to understand and implement the Queue data structure using arrays 
and circular queue logic in C.

## Technologies Used
- C Programming
- Data Structures
- Queue (Circular Queue Implementation)

## Features
- Initialize a queue
- Check if the queue is empty
- Check if the queue is full
- Add a call to the queue (Enqueue)
- Remove a call from the queue (Dequeue)
- Simulates a call center call handling system

## Data Structure Used
Queue (Circular Queue)

Queue follows FIFO principle:
First In → First Out

## Example Scenario
1. Caller 1 requests Technical Support
2. Caller 2 requests Billing Inquiry
3. Caller 3 requests Product Information

Calls are processed in the same order they arrive.
